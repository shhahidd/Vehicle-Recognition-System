# Vehicle Recognition System

A modern, fast, and responsive vehicle recognition dashboard built with React, Vite, and TailwindCSS.

## Features
- **Real-time Detection**: Integrates with Roboflow for vehicle detection.
- **License Plate Recognition**: Uses OCR to extract license plate numbers.
- **Dashboard**: View, filter, and manage detection logs.
- **Modern UI**: Dark mode support, glassmorphism effects, and responsive design.

## Getting Started

### Prerequisites
- Node.js (v18+ recommended)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd Vehicle-Recognition-System
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running the App
Start the development server:
```bash
npm run dev
```

### Building for Production
Create a production build:
```bash
npm run build
```

## Tech Stack
- **Frontend**: React, Vite, TailwindCSS
- **State/Icons**: Lucide React, Framer Motion
- **Backend/Database**: Superbase (Integrated)
- **AI/ML**: Roboflow, Tesseract.js (OCR)
